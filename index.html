Eco Scan

      As part of our human machine interface course, we have conceptualized an application in pairs from a given context : ecology.
EcoScan is based on a simple principle, offering users a scanner allowing them, by scanning a product, to obtain information related to its ecological impact, its origin, and more. As well as possible alternatives to this product. In addition, the application offers lessons and tips to allow everyone to become aware and learn more about ecology.

Loop Hero

      Loop Hero is a rogue-lite game based on the use of cards. The player have to position enemies, buildings and obstacles in the hero's path during his expedition and find the right balance so that he can survive and win a maximum of loot.
The aim of our school project was to make a simplified copy of this game in pairs using only Java.

"Cimetière" project

     In this project, the objective was to create a database allowing a municipality to manage its cemetery. All this by conceptualizing the database in relational database. Then by creating the tables under mySQL, by inserting an exploitable data sample. And finally, by conceptualizing a web site that exploits the database by implementing the web site with php.
